This repository contains a PostgreSQL database dump for the project "Build a Celestial Bodies Database" from freeCodeCamp.
The database models astronomical objects such as galaxies, stars, planets, and moons using relational tables and constraints.

Project Description
The goal of this project is to practice:
1. Designing relational database schemas
2. Using primary keys and foreign keys
3. Applying normalization
4. Writing SQL queries in PostgreSQL
